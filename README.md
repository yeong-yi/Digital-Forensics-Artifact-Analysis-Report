# 디지털 포렌식 가상 침해사고 분석 보고서

> 침해사고 발생 시 디지털 증거를 확보하고 분석하는 메커니즘을 학습하기 위해 가상의 윈도우 환경을 타깃으로 침해사고 시나리오를 직접 설계하고 생성된 이미지를 덤프하여 종합 디지털 포렌식을 수행한 개인 연구 프로젝트

---

## 프로젝트 개요
- **수행 기간:** 2024. 11. 30. ~ 2024. 12. 02. (개인 프로젝트)
- **주요 목표:** 가상 침해사고 환경에서 무결성이 확보된 증거를 수집하고, 시스템/어플리케이션 아티팩트 간의 연관 관계 분석 능력을 내재화함.
- **총 산출물:** 독자적인 디지털 포렌식 분석 보고서 (총 36페이지)
